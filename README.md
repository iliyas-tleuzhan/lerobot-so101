# LeRobot SO101 Autonomous System

## Overview
This project extends the LeRobot SO101 platform into an autonomous robotic system with perception, control, and actuation.

## Features
- Motor control and kinematics
- Autonomous task execution
- Modular software architecture

## Tech Stack
- Python / C++
- Embedded Controller (ESP32 / Arduino)
- OpenCV (planned)
- ROS2 (planned)
